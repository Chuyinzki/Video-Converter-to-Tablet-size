Put folders/files here (throw into the "Temp" folder for convenience).

Finds all .mp4, .mkv, and .avi videos inside the folder and subfolders.
Converts them to 720p MP4s with ~800 kbps video and 96 kbps audio bitrate.
Saves converted videos inside a new converted folder mirroring your original folder structure.
Skips anything already inside the converted folder so reruns do not reconvert output files.
Decodes in software for compatibility, but still uses your NVIDIA GPU for encoding.

Increase the PC's sleep timer if it's a big job.

Run:
.\convert_videos.ps1
